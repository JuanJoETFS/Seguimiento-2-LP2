¿Qué son las Listas y Colas?

Conceptos fundamentales de estructuras de datos en Python

📋 Listas (Arrays)

Una lista es una colección ordenada de elementos que puedes:

  • • Acceder por índice (posición)
  • • Modificar elementos existentes
  • • Agregar y eliminar elementos
  • • Almacenar diferentes tipos de datos
mi_lista = [1, 2, 3, "hola", True]

🔄 Colas (Queues)

Una cola sigue el principio FIFO (First In, First Out):

  • • El primer elemento en entrar es el primero en salir
  • • Ideal para procesamiento secuencial
  • • Usada en sistemas de atención al cliente
  • • Manejo de tareas en orden de llegada
from collections import deque

💡 Consejo: En este tutorial aprenderás a usar estas estructuras de datos con ejemplos prácticos que podrás aplicar en proyectos reales.

Listas en Python

Aprende a manipular colecciones de datos eficientemente

Operaciones Básicas con Listas

# Lista básica en Python
# Una lista es una colección ordenada de elementos

# Crear una lista
mi_lista = [1, 2, 3, 4, 5]
print(f"Lista original: {mi_lista}")

# Acceder a elementos (índice comienza en 0)
primer_elemento = mi_lista[0]  # 1
ultimo_elemento = mi_lista[-1] # 5
print(f"Primer elemento: {primer_elemento}")
print(f"Último elemento: {ultimo_elemento}")

# Modificar elementos
mi_lista[2] = 99
print(f"Lista modificada: {mi_lista}")

# Agregar elementos
mi_lista.append(6)  # Agregar al final
mi_lista.insert(0, 0)  # Insertar al inicio
print(f"Lista con elementos agregados: {mi_lista}")

# Eliminar elementos
mi_lista.remove(99)  # Eliminar por valor
elemento_eliminado = mi_lista.pop()  # Eliminar último elemento
print(f"Lista después de eliminaciones: {mi_lista}")
print(f"Elemento eliminado: {elemento_eliminado}")

Métodos Importantes

  • append() - Agregar al final
  • insert() - Insertar en posición
  • remove() - Eliminar por valor
  • pop() - Eliminar por índice
  • sort() - Ordenar lista
  • reverse() - Invertir orden

Propiedades Útiles

  • len(lista) - Longitud
  • lista[index] - Acceso
  • lista[-1] - Último elemento
  • lista[:] - Copiar lista
  • in - Verificar existencia
  • del - Eliminar elemento

Colas en Python

Implementación del principio FIFO con deque

Implementación de Cola

# Implementación de cola usando collections.deque
from collections import deque

# Crear una cola (FIFO - First In, First Out)
cola_supermercado = deque()

# Operaciones básicas de cola
# Encolar (agregar elementos al final)
cola_supermercado.append("Cliente 1")
cola_supermercado.append("Cliente 2")
cola_supermercado.append("Cliente 3")
print(f"Cola después de agregar clientes: {list(cola_supermercado)}")

# Desencolar (eliminar elementos del frente)
cliente_atendido = cola_supermercado.popleft()
print(f"Cliente atendido: {cliente_atendido}")
print(f"Cola restante: {list(cola_supermercado)}")

# Verificar si la cola está vacía
if not cola_supermercado:
    print("La cola está vacía")
else:
    print(f"Clientes en espera: {len(cola_supermercado)}")

# Ver quién está al frente sin eliminarlo
if cola_supermercado:
    print(f"Próximo cliente: {cola_supermercado[0]}")

Operaciones de Cola

  • append() - Encolar (add)
  • popleft() - Desencolar (remove)
  • clear() - Vaciar cola
  • len() - Tamaño de cola

Ventajas de deque

  • • Operaciones O(1) en ambos extremos
  • • Más eficiente que list para colas
  • • Thread-safe (seguro para hilos)
  • • Soporta operaciones adicionales

🎯 Importante: Usa collections.deque en lugar de listas para implementar colas, ya que es más eficiente.

Ejemplo Real: Sistema de Pedidos

Simulación de un sistema de gestión de pedidos para restaurante

Sistema de Gestión de Pedidos de Restaurante 

# Ejemplo real: Sistema de gestión de pedidos de restaurante
from collections import deque
import time
import random

class SistemaPedidos:
    def __init__(self):
        self.pendientes = deque()  # Cola de pedidos pendientes
        self.completados = []     # Lista de pedidos completados
        self.numero_pedido = 1
    
    def agregar_pedido(self, cliente, items):
        """Agrega un nuevo pedido a la cola"""
        pedido = {
            'numero': self.numero_pedido,
            'cliente': cliente,
            'items': items,
            'hora': time.strftime("%H:%M:%S"),
            'estado': 'pendiente'
        }
        self.pendientes.append(pedido)
        self.numero_pedido += 1
        print(f"📝 Pedido #{pedido['numero']} agregado para {cliente}")
        return pedido['numero']
    
    def procesar_siguiente_pedido(self):
        """Procesa el siguiente pedido en la cola"""
        if not self.pendientes:
            print("⚠️ No hay pedidos pendientes")
            return None
        
        pedido = self.pendientes.popleft()
        pedido['estado'] = 'completado'
        pedido['hora_completado'] = time.strftime("%H:%M:%S")
        self.completados.append(pedido)
        
        print(f"✅ Pedido #{pedido['numero']} completado para {pedido['cliente']}")
        return pedido
    
    def mostrar_estado(self):
        """Muestra el estado actual del sistema"""
        print(f"\\n📊 ESTADO DEL SISTEMA")
        print(f"🔄 Pedidos pendientes: {len(self.pendientes)}")
        print(f"✅ Pedidos completados: {len(self.completados)}")
        
        if self.pendientes:
            print("\\n📋 Próximos pedidos:")
            for i, pedido in enumerate(self.pendientes):
                print(f"   {i+1}. #{pedido['numero']} - {pedido['cliente']} ({len(pedido['items'])} items)")

# Simulación del sistema
sistema = SistemaPedidos()

# Agregar varios pedidos
sistema.agregar_pedido("Ana", ["Hamburguesa", "Papas fritas", "Refresco"])
sistema.agregar_pedido("Carlos", ["Pizza familiar", "Ensalada"])
sistema.agregar_pedido("María", ["Sushi", "Sopa miso"])
sistema.agregar_pedido("Juan", ["Tacos", "Agua de horchata"])

# Mostrar estado inicial
sistema.mostrar_estado()

# Procesar algunos pedidos
print("\\n🍳 Procesando pedidos...")
sistema.procesar_siguiente_pedido()
sistema.procesar_siguiente_pedido()

# Mostrar estado final
sistema.mostrar_estado()
